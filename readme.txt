Accept cryptocurrency on your WooCommerce stores through CoinMall.com


== Description ==

CoinMall allows anyone with a WooCommerce store to instantly start accepting cryptocurrency payments. Not only that, but the plugin also supports Bitcoin Lightning Network payments, as well as the ability for CoinMall.com users to pay with their site balance directly.

== Installation ==

1. Upload the `coinmall` directory to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. In the WooCommerce Settings page go to the Payment Gateways tab, then click CoinMall.
1. Check "Enable CoinMall" and enter your CoinMall API public key and secret key.
1. If you haven't already done so, sign up on CoinMall.com to get an API public and secret key.
1. Click "Save changes" and the gateway will be active.

== Frequently Asked Questions ==

= Does CoinMall take any fees? =

No fees are taken whatsoever. The amount the buyer pays, the seller receives.

= What cryptocurrency do you currently support? =

CoinMall currently supports Bitcoin and ZCash, however we are continuously working on adding more value-add cryptocurrencies to our selection.
